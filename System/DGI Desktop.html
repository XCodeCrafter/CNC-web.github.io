DGI Desktop CNC Simulator

ChipBoss™ Software

DGI Desktop is the Milltronics 9000 Series CNC control software that you can run on your PC. DGI Desktop allow you to:

  • Learn the control on your PC (the Desktop Software simulates the software on the control)
  • Experience the full capability of conversational programming Develop complex programs using Milltronics conversational or G code programming
  • Realize the efficiency of solid model program verification and part visualization capabilities
  • Verify programs and see the graphic of the part appear on the screen as the tool cuts away stock
